### 프로젝트 개요

적의 탄막을 피하면서 최대한 버티는 엔드리스 3D Top-View 슈팅 게임입니다.

Unity로 구현한 프로젝트입니다.

### 구현 목록

- 적의 공격 기능 구현 및 플레이어 상태(이동, 대시, 피격)를 구현하였습니다.
- 적 객체 및 탄막을 미리 생성 및 리스트로 관리하여 객체 생성으로 인한 프리징 현상을 해결하였습니다.
- WebGL로 빌드하여 웹 페이지에 배포하였습니다.
